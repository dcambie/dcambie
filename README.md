Far too often academic code is either closed sourced or "_available upon [reasonable] request_".

I find statements like this in contrast with the mission of academic research to further human knowledge. Such attitude hinders reproducibility and negatively affect the scientific community as a whole.


I truly believe in open science, therefore I pledge to:
 * release publicly the raw data and source code associated with the publications generated in my research group _in a reusable format_ unless legally prevented to do so by NDA or other IP agreements.
 * support open-access in a pragmatic way, e.g. publishing preprints for manuscripts submitted to non open-access journals.
